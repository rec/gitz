``git infer``: Commit changes with an automatically generated message
---------------------------------------------------------------------


(from https://github.com/moondewio/git-infer)
USAGE
=====

.. code-block:: bash

    git infer [-a] [-A] [-h]
    
       -a    Include all tracked files
       -A    Include all files, even untracked ones
       -h    Show this help message
