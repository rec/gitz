``git st``: Colorful, compact git status
----------------------------------------

USAGE
=====

.. code-block:: bash

    git st

DESCRIPTION
===========

One-line status for each file with lines added and removed.

This version written by https://github.com/PlatyPew/,
original version by https://www.reddit.com/user/ex1c)
